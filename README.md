#Tripvise

##Getting Started

run `script/bootstap`

configure `config/database.yml` and `config/application.yml` if necessary.

##Tests

If you're setting up this app, you need to do one thing before running the specs:

`bin/rake db:setup`

To run all specs:
`bin/rspec`
